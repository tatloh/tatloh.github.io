ListsUsage
Unordered ListAn unordered list in HTML will show a list of items not in any specific order. Each item will be shown with a bullet symbol in front by default. You could use an unordered list to list the different colors avialable for a car model for sale for examnple. Here are the tags you would use to display an unordered list: <ul><li>item1</li><li>item2</li></ul>
Ordered ListOrdered lists will present items in a specific order and by defult will be ordered by numbers. The tags used for ordered lists will be: <ol><li>item1</li><li>item2</li></ol>
